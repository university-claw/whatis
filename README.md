# 명지클로 소개 페이지

GitHub Pages에 그대로 배포할 수 있는 정적 홍보 페이지입니다.

## Local Preview

```bash
cd whatis
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## GitHub Pages

This directory is ready to be used as the repository root for GitHub Pages. Push it to `main`, enable GitHub Pages with the Actions source, and `.github/workflows/pages.yml` will deploy the static site.
